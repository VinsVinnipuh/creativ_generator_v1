# Telegram бот для генерації Instagram креативів

Цей проєкт — готова основа Telegram-бота на Python, який:

1. Приймає від користувача бриф (публічну інформацію про продукт/послугу).
2. Приймає візуальні референси (фото).
3. Генерує кілька варіантів креативів для Instagram за допомогою OpenAI API.

## Можливості

- Команди:
  - `/start` — коротка інструкція.
  - `/new` — очищення поточного брифу та референсів.
  - `/generate` — генерація креативів на основі зібраного контексту.
- Підтримка мультимодального вводу:
  - Текстові повідомлення додаються до брифу.
  - Фото в Telegram додаються як референси (через file URL).
- Налаштування кількості варіантів через змінну `CREATIVE_VARIANTS`.
